# grid-probrem
在 N×N 的表格中，每格填入 1 或 -1，使得每格的數字剛好是相鄰格子內數字的乘積，（這裡的相鄰指的是上下左右），請問有幾種填滿表格的方法？
## 變數說明
|   變數名稱   |          代表意義          |    實際範例    |
|----------|------------------------|------------|
| width    | 題目中的N（方形表格寬度）          | 7          |
| modalNum | 可能會符合規範的模型數（2的width次方） | 128（2的7次方） |
